# 抽真空灌注段三维可视化系统 - 设计规格

## 1. 概念与愿景

一个专业级的B端三维可视化监控系统，采用全息投影美学呈现抽真空灌注工艺段的实时状态。整体风格为通透的科技蓝调，搭配半透明毛玻璃效果和柔和的蓝光晕染，营造出专业、高效、现代的工业监控体验。用户仿佛透过全息投影界面直观掌控整个灌注工艺。

## 2. 设计语言

### 美学方向
- **风格**: 全息投影风格 - 半透明毛玻璃 + 科技蓝光 + 全息网格背景
- **参考**: B端专业监控界面 + 现代数据可视化 + 航空全息显示屏
- **特点**: 通透、轻盈、专业、高效

### 字体
- 标题: "Orbitron", "Rajdhani", sans-serif (科技感)

- 正文: "Inter", "Noto Sans SC", sans-serif

### 动效哲学
- 仪表盘数据: 数字滚动动画，模拟真实数据刷新
- 按钮交互: 按下时有轻微发光反馈 + 悬停缩放
- 三维模型: 持续的缓慢旋转，环境光效呼吸
- 状态切换: 平滑的渐变过渡 (300ms ease-out)
- 指示灯: 柔和的脉冲发光效果表示活跃状态
- 全息网格: 静态背景，营造科技氛围
- 扫描线: 极其微弱的扫描线效果，不干扰阅读

### 视觉特效
- **毛玻璃效果**: backdrop-filter: blur(20px) 实现半透明毛玻璃
- **全息网格**: 40px间隔的淡蓝色网格线
- **发光边框**: 半透明发光边框营造科技感
- **阴影**: 柔和的彩色阴影，避免生硬
- **透明度层次**: 不同组件使用不同透明度区分层次

## 3. 布局结构

### 页面尺寸
- 设计尺寸: 1920 x 1080px

### 左侧面板内容
- **设备状态列表**: 真空泵、灌注泵、各阀门开闭状态（指示灯形式）
- **实时数据**: 真空度、流量、压力、温度的简略数值显示
- **报警信息**: 最新报警记录列表

### 右侧面板内容
- **工艺参数监控**: 完整的参数仪表盘和详细数值
- **控制按钮组**: 开始/暂停/停止灌注、泄压、抽真空、复位
- **工艺进度**: 当前阶段、进度条、剩余时间

### 顶部状态栏 (70px)
- Logo + 工位名称
- 实时时钟
- 运行状态 (运行中/暂停/停止)
- 报警计数

## 4. 功能与交互

### 三维可视化
- 使用Three.js加载并显示GLB格式三维模型
- 支持鼠标拖拽旋转、滚轮缩放、右键平移
- 自动旋转模式（可开启/关闭）
- 环境光效和点光源营造立体感

### 数据监控
- 实时数值动态变化（每100ms更新）
- 仪表盘进度条平滑动画
- 设备状态指示灯实时反馈
- 报警记录自动滚动更新

### 控制按钮
- **开始**: 启动灌注流程，状态变为"运行中"
- **暂停**: 暂停当前流程，状态变为"已暂停"
- **停止**: 停止流程，状态变为"已停止"
- **抽真空**: 启动真空系统
- **泄压**: 释放系统压力
- **复位**: 重置所有参数到初始状态

### 报警系统
- 实时监测各项参数阈值
- 超过阈值时触发报警，红色闪烁提示
- 报警记录自动追加到列表
- 支持同时显示多条报警

## 5. 组件清单

### 状态指示灯 (meta-dot)
- **.on**: 绿色发光，表示正常运行
- **.off**: 红色，表示停止/故障
- **.idle**: 灰色，表示待机

### 仪表盘 (gauge-unit)
- 圆弧进度条显示数值占比
- 中心显示当前数值和单位
- 颜色根据参数类型变化

### 控制按钮 (ctrl-btn)
- 毛玻璃背景 + 发光边框
- 悬停时边框增强发光
- 激活时背景色变化
- 支持.start/.pause/.stop/.vacuum/.reset样式

### 数据卡片 (metric-card)
- 悬停时轻微上浮 + 边框发光
- 顶部有淡色装饰线

### 报警条目 (alarm-entry)
- 左侧红色竖条标识
- 包含时间和描述

## 6. 技术实现

### 前端框架
- 原生HTML5 + CSS3 + JavaScript
- Three.js (r128) 用于三维渲染
- GLTFLoader 用于加载GLB模型

### 关键库
- Three.js: 3D渲染核心
- OrbitControls: 相机控制
- GLTFLoader: 模型加载

### 性能优化
- requestAnimationFrame驱动动画循环
- 合理的重绘区域控制
- 仪表盘使用CSS过渡而非JS动画
- 粒子效果使用CSS动画
